# Sound Effects for Bingo App

This directory contains sound effect files for the Bingo application.

## Required Sound Files

Please add the following MP3 files to this directory:

1. **drum_roll.mp3** - Played during roulette spin (approximately 2 seconds)
2. **draw_number.mp3** - Played when the host draws a new number
3. **reach.mp3** - Played when a player achieves reach
4. **bingo.mp3** - Played when a player gets bingo

## Sound Sources

You can obtain royalty-free sound effects from:
- [Freesound](https://freesound.org/)
- [Zapsplat](https://www.zapsplat.com/)
- [Mixkit](https://mixkit.co/free-sound-effects/)

## License Requirements

Ensure all sound files are royalty-free or properly licensed for commercial use.

## File Specifications

- Format: MP3
- Sample Rate: 44.1kHz or 48kHz
- Bit Rate: 128kbps or higher
- Duration: 0.5-2 seconds (short and punchy)

## Placeholder Files

Until actual sound files are added, the application will fail silently and log warnings to the console when attempting to play sounds.
